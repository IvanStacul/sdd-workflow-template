# Skill Registry - Skills de Proyecto

> Skills locales que no son parte del flujo SDD pero aportan convenciones
> de stack, framework o dominio. El skill-resolver las inyecta compactadas
> en fases técnicas cuando un change las necesita.

| Skill | Tipo | Inyectar cuando |
|-------|------|-----------------|
| `interface-design` | `styling` | El change toca UI, componentes, layouts o dashboards |
| `vercel-react-best-practices` | `frontend` | El change toca componentes React o páginas Next.js |
| `visual-companion` | `general` | La fase necesita mostrar mockups, diagramas o comparaciones visuales |
