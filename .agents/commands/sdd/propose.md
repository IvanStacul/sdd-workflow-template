---
name: "SDD: Propose"
description: "Crear un nuevo change con propuesta formal"
---

Crear un nuevo change. Si el usuario no proporciono nombre, preguntar.

Invocar la skill `sdd-propose` con el nombre y la descripción del change.
Si la skill clasifica el análisis cruzado como `obligatorio` o `recomendado`, debe crear o actualizar `impact-map.md`; si queda `opcional`, debe justificar la omisión.
